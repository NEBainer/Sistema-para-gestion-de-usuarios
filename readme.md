# User Management System

Aplicación web desarrollada con JavaScript Vanilla, Bootstrap 5 y LocalStorage.

---

## 🚀 Funcionalidades

- Listado de usuarios desde API externa  
- Alta de usuarios  
- Modificación de usuarios  
- Eliminación de usuarios  
- Persistencia mediante LocalStorage  
- Validación de formularios  
- Alertas dinámicas  
- Exportación de datos a CSV  

---

## 🛠️ Tecnologías utilizadas

- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript ES6+  
- Fetch API  
- LocalStorage  

---

## 🎯 Objetivo

Proyecto desarrollado como práctica de CRUD completo para afianzar conceptos de manipulación del DOM, asincronismo, consumo de APIs y persistencia de datos en el navegador.
